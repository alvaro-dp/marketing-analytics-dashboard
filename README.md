# Marketing Analytics & Campaign Optimization Dashboard

## 📌 Business Problem

Many companies invest heavily in digital campaigns but lack visibility into:

- unprofitable campaigns
- budget waste
- scalable segments
- performance anomalies
- ROI optimization opportunities

This leads to inefficient investment allocation and poor strategic decision-making.

---

## 🎯 Objective

Build an executive-level Marketing Analytics system focused on:

- profitability analysis
- campaign optimization
- anomaly detection
- scalable opportunities
- strategic decision-making

---

## 🏗️ BI Architecture

This project implements a professional BI architecture using:

- Power Query for ETL
- Star Schema data modeling
- DAX recalculated KPIs
- Business rules & alerts
- Executive-level KPI monitoring

---

## 📊 Key KPIs

- ROAS
- CPA
- CPC
- CTR
- Profit
- Conversion Rate

---

## 🚨 Business Rules

The dashboard detects problematic campaigns using rules such as:

- Low ROAS
- High CPC
- Expensive campaigns
- Unprofitable segments

---

## 🔍 Key Insights

The system allows identification of:

- budget waste
- profitable campaigns
- scalable opportunities
- performance deterioration
- optimization opportunities

# BI Architecture

This project implements a scalable Business Intelligence architecture designed for executive-level Marketing Analytics.

---

# Data Source

The project uses marketing performance datasets containing:

- impressions
- clicks
- conversions
- ad spend
- revenue

The architecture is designed to support future CRM or Dozer integrations.

---

# ETL Layer

Power Query was used for:

- data cleaning
- standardization
- type validation
- removal of precalculated metrics
- transformation and normalization

---

# Data Modeling

A Star Schema model was implemented to improve:

- scalability
- performance
- maintainability
- multidimensional analysis

---

# DAX Layer

KPIs were recalculated using DAX measures to ensure:

- consistency
- centralized business logic
- accurate calculations

---

# Visualization Layer

The dashboard focuses on:

- profitability analysis
- anomaly detection
- executive KPI monitoring
- campaign optimization
